## 📋 About Nexura

**Nexura** is a state-of-the-art, empathetic chronic care management platform designed to track vital health logs, sleep patterns, and medication adherence while analyzing medical reports using multimodal AI intelligence.

### ✨ Key Features
- **Strict Authentication Flow**: Seamless and secure onboarding flow (`Sign Up` ➔ `Log In` ➔ `Dashboard`).
- **Empathetic AI Companion**: Real-time conversational health analysis tailored to chronic care patients.
- **Medication & Adherence Tracking**: Schedule alerts and track compliance rates.
- **Sleep & Vital Logging**: Monitor consistency and physiological degradation trends over time.
- **Emergency Caregiver Escalation**: Automated alerts dispatched to caregivers during critical situations.

## 💻 Running Locally

If you prefer to run the full Spring Boot + PostgreSQL backend locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rajeswari168/Nexura---Hackathon-Project.git
   cd Nexura---Hackathon-Project
   ```

2. **Run the Spring Boot application:**
   ```bash
   ./mvnw spring-boot:run
   ```

3. **Open the application:**
   Navigate to **[http://localhost:8080/index.html](http://localhost:8080/index.html)** in your web browser.
